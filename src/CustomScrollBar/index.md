> 使用方式,参考组件 CustomScrollBar

```ts
const { dragY, onDragStart } = usePortraitDrag(height - barHeight);

<div className={getCls('wrapper')}>
  <div
    className={getCls('yq')}
    onMouseDown={(e) => {
      onDragStart(e);
    }}
    style={{
      height: barHeight,
      top: dragY,
    }}
  ></div>
</div>;
```

## api

| option          | description | type                | required | example | remark |
| --------------- | ----------- | ------------------- | -------- | ------- | ------ |
| children        |             | any                 | true     |         |        |
| height          | 容器的高度  | number              | true     |         |        |
| scrollSpeed     |             | string              | false    |         |        |
| squareStyle     | 滑块样式    | React.CSSProperties | false    |         |        |
| barWrapperStyle | 滚动框样式  | React.CSSProperties | false    |         |        |
